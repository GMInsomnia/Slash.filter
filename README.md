Overview

Slash.filter is an intelligent, progressive loot filter for Project Diablo 2, designed to streamline farming, RuneWords, and crafting.

Key Features

Progressive Hide System

Automatically hides less important items, highlighting only valuable ones.

Works across all item categories (Normal, Magic, Rare, Unique, Set).

BASE + Meta Info

Displays key item data: ilvl, price+, and socket count.

Helps quickly evaluate potentially valuable drops.

Rune Alerts / DropNote

Notifications for dropped runes, RuneWords, and rare items.

Adjustable by filter level (Relaxed, Medium, Strict).

Crafting & Recipes

Shows all crafting recipes and components, including the Horadric Cube.

Simplifies gathering items for crafting.

Lore Info Settings

Optionally display lore info on drop.

Improves inventory management during farming.

Filter Levels

Relaxed, Medium, Strict

Potionless (Strict), LLD Bases (Strict)

Show Recipes (Relaxed)

Authorship & Inspiration

Slash.filter uses the standard PD2 syntax and structure, inspired by eqN, BH, and Kryszard.

All unique features — Progressive Hide, BASE/meta info, Rune Alerts, and Crafting/Recipe support — are original and not found elsewhere.

This filter combines familiar ideas with innovative functionality, making it truly unique.

Usage Instructions

Copy slash.filter into your Project Diablo 2 filter directory.

Launch the game via the PD2 launcher and select Slash.filter.

Choose your preferred filter level: Relaxed / Medium / Strict / Show Recipes.

Enjoy a cleaner inventory, visible RuneWords, and streamlined crafting collection.
